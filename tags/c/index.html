a simple d-bus daemon implemented with c

2014-06-04
#c  #programming 
first of all, you need a d-bus config file for the daemon. on a standard debian system it is saved in /etc/dbus-1/system.d/ as cneod.conf. and here you can find the daemon code. what can you do with the compiled code? with the d-bus rules above, you are able to: shutdown suspend reboot setbrightness getbrightness your system without root access. if you have a closer look to the code, you will see, the program is intended for my openmoko GTA02 and for my laptop/netbook.
