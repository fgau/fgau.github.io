building a budget-friendly kubernetes cluster

2024-10-27
#hardware 

turn four old dell laptop motherboards into a mini computing cluster with ubuntu! here’s how:

  • prepare each node: strip down the laptops to essential parts and install ubuntu server. set up unique hostnames and static IPs
  • connect nodes: link each to a network switch using ethernet cables
  • i have used these excellent blog article to setup control-plane and nodes
  • to get really deep insights into your cluster, k9s is the tool of choice

this project is perfect for learning about clusters and giving old laptops a second life.
